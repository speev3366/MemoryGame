# Memory Duel V7.5.9

Тази версия затяга онлайн режима и auth потока:
- по-стабилен login без автоматично прехвърляне към онлайн лоби при обикновен вход
- по-добър invite flow за guest / profile вход
- waiting room-и без втори играч се считат за изтекли след 30 минути
- по-стабилно създаване / затваряне / подмяна на стаи
- по-стабилен старт на online match след invite join
- по-стабилен layout на top bar-а по време на игра

## Ако проектът е нов
1. Отвори `supabase-schema.sql` в Supabase SQL Editor.
2. Натисни `Run`.
3. Попълни `supabase-config.js`.
4. След това пусни и `supabase-migration-v7.5.9.sql`.

## Ако вече имаш Supabase проект
1. Отвори `supabase-migration-v7.5.9.sql` в Supabase SQL Editor.
2. Натисни `Run`.
3. Качи новите файлове в GitHub.
4. Направи `Ctrl + F5` или отвори сайта в Incognito.

## Онлайн поток
1. Host влиза в профила и отваря `Онлайн`.
2. Създава стая:
   - свободна
   - с парола
   - с invite линк
3. Вторият играч влиза:
   - от публичното лоби
   - с код
   - с парола
   - с invite линк
4. При invite линк гостът може да продължи и като guest, ако Anonymous Auth е включен.
5. Домакинът натиска `Старт онлайн`.
