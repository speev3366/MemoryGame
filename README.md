# Memory Duel V7.3

Тази версия добавя:
- профил и auth от икона горе вдясно
- гост режим за локална игра с автоматични имена `GuestMemory######`
- онлайн режимът е заключен, докато потребителят не влезе
- автоматичен вход веднага след успешна регистрация
- `Запомни ме` при логин
- waiting room / lobby със:
  - публични стаи
  - стаи с парола
  - invite стаи с линк
- профил панел с история на игрите
- запис на локални, AI и онлайн мачове в `match_history` за влезлия потребител

## Ако проектът е нов
1. Отвори `supabase-schema.sql` в Supabase SQL Editor.
2. Натисни `Run`.
3. Попълни `supabase-config.js`.

## Ако вече имаш стар Supabase проект
1. Отвори `supabase-migration-v7.3.sql` в Supabase SQL Editor.
2. Копирай съдържанието му и натисни `Run`.
3. Качи новите файлове в GitHub.
4. Направи `Ctrl + F5` или отвори сайта в Incognito.

## Нов онлайн поток
1. От иконата горе вдясно влизаш или се регистрираш.
2. Избираш режим `Онлайн`.
3. Създаваш стая:
   - свободна
   - с парола
   - с invite линк
4. Другият играч влиза от waiting room-а, с код, с парола или с invite линка.
5. Домакинът натиска `Старт онлайн`.

## Бележка за гостите
Гостите не се записват в базата и не могат да отварят онлайн стаи. Това е правилен компромис:
- локалната игра остава бърза
- базата пази само реални профили и реална история
- няма шум от временни guest акаунти
